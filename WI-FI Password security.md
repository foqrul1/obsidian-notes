Most of the wifi connection are secure. there are several protocol to use wifi to secure. Lets discuss about these protocol

### WEP (Wired Equivalent Privacy)
very first wireless security protocol. Later found a lot of vulnerability and also found 40 bit encryption key is not enough secure. and it's easily hackable.
### WPA (wi-fi protected access)
Stronger encryption. Uses TKiP(Temporary key integrity protocol). but later TKIP also find lots of vulnerability.

### WPA2 (wi-fi protected access)
provides even stronger security than WPA. Uses AES(advanced encryption standard). Adopted by the U.S. govt.

### WPA3
The next generation of wireless security. introduced in 2018. received increased security protection from password gue
